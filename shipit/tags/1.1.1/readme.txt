== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/shipit` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. Create a tab called `Settings Shipit`, where you must add your user granted by Shipit.
5. Ready is already integrated with the shipit quote and management of your orders.
